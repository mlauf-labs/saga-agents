# AGENTS.md

The full guidance for AI coding agents (and humans) working in **SAGA Agents** lives in
**[CLAUDE.md](CLAUDE.md)**.

👉 **Read [CLAUDE.md](CLAUDE.md) before making any changes.** It covers the project overview,
tech stack, repository layout, commands, how agents are defined (`config/agents/*.md`), key

coding conventions, and the Git commit / branch / pull-request workflow (`main` and `develop`
are protected — use a feature branch + PR).
